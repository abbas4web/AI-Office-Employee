# AI Office Employee - Frontend

Simple React frontend for AI Office Employee application.

## Quick Start

1. **Install dependencies**
   ```bash
   cd frontend
   npm install
   ```

2. **Start development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```

## Pages

- **Login** (`/login`) - Simple login page
- **Dashboard** (`/dashboard`) - Overview with stats
- **Tasks** (`/tasks`) - Task management (CRUD)
- **Clients** (`/clients`) - Client management (CRUD)

## Features

- React functional components
- React Router for navigation
- Clean, minimal CSS
- API integration with deployed backend
- Responsive design

## API Endpoints

The frontend connects to the deployed backend at:
`https://ai-office-employee-api.vercel.app`

## Technology Stack

- React 18
- Vite (build tool)
- React Router 6
- Plain CSS (no UI libraries)
